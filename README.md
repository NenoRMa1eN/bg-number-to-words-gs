# bg-number-to-words-gs 🇧🇬

Google Apps Script библиотека за преобразуване на числа и парични суми в думи (словом) на български език за **Google Sheets**.

Изключително подходяща за фактуриране, приемо-предавателни протоколи, договори и финансови справки.

---

## 🌟 Основни предимства

- **Граматическа прецизност:** Правилно определяне на граматичния род за хиляди (*„две хиляди“*), милиони и валути.
- **Интелигентен съюз „и“:** Вградена логика за правилно поставяне на съюза „и“ според правилата на българския език (*„един милион и хиляда“*, *„сто и пет“*).
- **Прецизна обработка на дробната част:** Избягва типичните floating-point проблеми на JavaScript.
- **Разширяема архитектура:** Лесно добавяне на нови валути и мерни единици.
- **Автодовършване (JSDoc):** Вградени подсказки директно в интерфейса на Google Sheets при писане на формулите.

---

## 🚀 Инсталиране

1. Отвори таблицата си в **Google Sheets**.
2. Избери **Разширения → Apps Script**.
3. Изтрий примерния код.
4. Копирай съдържанието на `numberToWordsBG.gs` в редактора.
5. Запази проекта.
6. Върни се в Google Sheets – функциите вече са готови за използване.

---

## 📚 Публичен API

| Функция | Предназначение |
|---------|----------------|
| `BGLEV()` | Сума в левове |
| `BGEURO()` | Сума в евро |
| `LEV()` | Кратък псевдоним на `BGLEV()` |
| `EVRO()` | Кратък псевдоним на `BGEURO()` |
| `inWordsBG()` | Произволна парична единица |
| `numberToWordsBG()` | Цяло число с произволна мерна единица |

---

## 💻 Примери

### Левове и евро

```gs
=BGLEV(123.45)
```

→ **сто двадесет и три лева и четиридесет и пет стотинки**

```gs
=BGEURO(2000)
```

→ **две хиляди евро**

```gs
=LEV(1001000)
```

→ **един милион и хиляда лева**

> В зависимост от регионалните настройки на Google Sheets разделителят между аргументите може да бъде `,` или `;`.

### Произволни валути

```gs
=inWordsBG(12.50; "долар"; "цент")
```

→ **дванадесет долара и петдесет цента**

```gs
=inWordsBG(100; "паунд"; "пенс")
```

→ **сто паунда**

### Числа с мерни единици

```gs
=numberToWordsBG(2; "литър")
```

→ **два литра**

```gs
=numberToWordsBG(25; "страница")
```

→ **двадесет и пет страници**

```gs
=numberToWordsBG(25; "година")
```

→ **двадесет и пет години**

---

## ⚠️ Ограничения

- Поддържат се числа до **999 999 999**.
- `inWordsBG()` приема, че дробната част е **1/100** от основната единица и е предназначена основно за парични суми.

---

## ⚙️ Добавяне на нови валути и изключения

Добави нов запис в `BG_IRREGULAR_UNITS`:

```javascript
const BG_IRREGULAR_UNITS = {
    // ...

    "франк": {
        one: "франк",
        other: "франка"
    }
};
```
